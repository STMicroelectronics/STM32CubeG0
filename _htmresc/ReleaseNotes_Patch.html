STM32CubeG0 Firmware Package V1.6.3 / 27-February-2026

Main Changes

  • General updates to fix known defects and enhancements implementation.
  • SLA0044 Legal terms updated to latest revision (Rev 6 / October 2025)
  • SLA0048 Legal terms updated to latest revision (Rev 5 / October 2025)
  • Add the GPIO_Demo example for all supported STM32G0xx Nucleo boards.
    • This example provides the source code for the demonstration binary package, available for download on the specific Nucleo webpage.
  • Deliver SW_Security_Level.md file to state that STM32CubeG0 is classified to Security Level Low.
  • Upgrade USBPD Devices library to use v3.5.2
  • Upgrade USB Host library to use V3.5.3.
  • Upgrade USB Device library to use V2.11.2.
  • Update power voltage monitoring to align with the new STM32G0B1xx/C1xx product documentation (RM0444 Rev6)
  • Upgrade HAL drivers to be compliant with Synopsys Coverity static analysis tool.
